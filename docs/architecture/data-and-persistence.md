# Data and Persistence

Persistence in LeanKernel centers on sessions, turns, diagnostics, and supporting metadata.

## Key Responsibilities

- `LeanKernel.Persistence`: session and turn durability.
- `LeanKernel.Diagnostics`: diagnostics storage and retrieval.
- `LeanKernel.Knowledge`: retrieval-backed knowledge integration.
